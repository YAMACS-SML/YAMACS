This folder contains video tutorials showing how YAMACS works. The folder will be updated as new plugins are created.


10/05/22 - New tutorials upload:
- apoprotein_all_plgs shows all the plugins on an apoprotein system
- md_play_plg shows the MD Play plugin, one of the main functionalities of YAMACS, which allows to directly load and visualize the results of GROMACS dynamics
- select_atom_plg shows the atom selection process, which allows the user to perform analyses on the selected atoms using GROMACS engines; currently the following analyses are available: RMSD, RMSF, Rg and SASA.


NOTE: videos are still work in progress and are also being uploaded on our YouTube channel. The link will be posted as soon as the upload is finished.
